# Color & Materials

A native iPhone color and material system for hierarchy, meaning, and depth. Start with system backgrounds, semantic colors, and Liquid Glass-compatible surfaces. Add brand color sparingly.

## When to Use

- **Hierarchy**
- **Meaning**
- **Depth**
- **Emphasis**

## Design System Rules

### Do

- Start with semantic system colors
- Use tint for meaning, not decoration
- Let materials handle depth where appropriate
- Keep the palette restrained

### Don't

- Paint every surface with a custom color
- Fight system materials with heavy custom chrome
- Use color as the only status signal

## Common Patterns

```swift
Text("Title")
    .foregroundStyle(.primary)
```
